# RainOfCubes_20

Продолжаем работу над задачей “Дождь кубов” (RainOfCubes_10). 
Реализовано несколько платформ  (под наклоном несколько меньших размеров над главной). 
После удаления куба, на его месте создается бомба (черный шар с коллайдером и rigidbody), которая плавно становится прозрачной (у каждой бомбы выбирается время от 2 до 5 секунд) и затем взрывается (раскидывает объекты в указанном радиусе). 

Для реализации эффекта затухания в используемом Material был выбран режим рендеринга Fade. Данный режим позволяет корректно изменять значение альфа-канала материала, обеспечивая плавное появление или исчезновение объекта
Бомбу, как и куб, создает спавнер.
Для создания похожих спавнеров изучена тема Обобщения (Generics).

Каждый спавнер имеет показатель для отображения следующих параметров на экране:
1) количество заспавненых объектов за всё время
2) количество созданных объектов 
3) количество активных объектов на сцене
Информация показывается отдельно кубов и отдельно бомб
